========================================
revo fit — はじめにお読みください
========================================

このフォルダは revo fit（フィットネスアプリ）の
完成版デザイン一式です。Claude Code での実装に使います。

■ Windowsでの開き方
1. このZIPを右クリック →「すべて展開」
2. 展開された「revo_fit」フォルダを開く
3. screens フォルダ内の .html を
   ダブルクリックすればブラウザで各画面を確認できます

■ まず読むファイル（順番）
1. README.md          … 全体説明・実装の鉄則
2. SCREENS.md         … 全56画面の索引
3. docs/01_onboarding_flow.md   … 起動フロー（重要）
4. docs/02_log_to_home_sync.md  … 記録→ホーム連携（重要）
5. ROADMAP.md         … 申請までの進め方

■ フォルダの中身
revo_fit/
  README.md           … 説明書
  SCREENS.md          … 画面索引
  ROADMAP.md          … ロードマップ
  START_HERE.txt      … このファイル
  docs/               … 設計書（実装の要）
  screens/            … 全56画面 ＋ BGM音源(City_Pulse.mp3)

■ Claude Code への渡し方
このフォルダごと Claude Code に渡して、
「README.md の鉄則を守って、screens/ のモックを
 忠実に実装して」と伝えてください。
docs/ の2つの設計書（フローと連携）も必ず読ませてください。

■ 注意
・.md ファイルはメモ帳でも開けますが、
  文字が見やすいのは VS Code や Typora などです。
・画面の .html は必ずブラウザ（Chrome/Edge）で開いてください。
