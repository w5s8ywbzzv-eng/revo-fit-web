# revo fit — 実装引き継ぎ（Claude Code 用）

このフォルダは、**revo fit（フィットネス／ウェルネスアプリ）** の **UIモックアップ（HTML/CSS/JS 単一ファイル）** 一式です。デザイン・文言・画面遷移の設計が固まった状態で、ここから実装（本番のアプリ化）に入れます。

> ⚠️ このZIPは **revo fit のユーザー向け画面のみ** を収録しています。運営管理（admin）・revo life・ランディングページ（LP）は **含みません**（別管理）。

---

## ★ 実装の鉄則（UXを壊さないために・最優先で読むこと）

過去、実装時にモックのUX/デザインが崩れた経緯があります。以下を厳守してください。

1. **各HTMLモックが「正」。見た目・余白・文言・遷移・アニメーションを勝手に変えない。** 実装は「このモックをそのまま動かす」のがゴール。デザインの再解釈・簡略化・"改善"は禁止。変更が必要なら必ず理由を添えて確認を取る。
2. **デザイントークンは各ファイル内の `T`（テーマ）オブジェクトに集約されている。** 色・角丸・タイポはハードコードで散らさず、必ずトークン経由で。黒×金（#E3C56A）が基準。ライトテーマも全画面対応済みなので、両テーマを維持する。
3. **文言（コピー）は一字一句そのまま使う。** LP・アプリ内のコピーは景表法・薬機法を意識して調整済み。特に食事まわり等の「誤解を招かない」言い回しを勝手に書き換えない。
4. **多言語は14言語。** 各画面のJS内 `langs` / 翻訳オブジェクトを踏襲。日本語前提のレイアウト崩れ（英・独の長文）に注意。
5. **サンプル数値は必ずDB取得値に置換し、新規ユーザーには「ゼロ」状態（エンプティステート）を用意する。** モックの数字をそのまま焼き込まない。詳細はROADMAP 13章。
6. **アニメーションの塩梅（登場の間・光の演出など）はモックのCSSを再現する。** 派手にしすぎない／静かすぎない、の調整済みバランスを保つ。
7. 迷ったら「モックがどう見えるか」を基準に。実装の都合でUXを削らない。

## 1. これは何か

- 各 `.html` は **単一ファイルで完結するUIモック**（外部依存は Tabler Icons と Google Fonts の CDN のみ）。
- そのままブラウザで開けば動作を確認できます（言語切替・テーマ切替・各種インタラクションは JS で実装済み）。
- **データはすべてハードコードのサンプル**です。実際のDB連携・課金・認証は未実装（下記「未実装」参照）。
- ブランド：**revo**（小文字）。シリーズ展開：**revo fit / life / running / world**。
- テーマ：ダーク（黒×金 #E3C56A）が基準。ライトテーマも全画面対応。
- 多言語：ユーザー向け画面は **14言語**（日本語/English/简体/繁體/한국어/Français/Español/Deutsch/Italiano/Português/ภาษาไทย/हिन्दी/Tiếng Việt/Bahasa Indonesia）。運営管理画面は **日本語のみ**。

## 3. 実装済み（このモックで確定していること）

### ブランド／デザイン
- **アプリアイコン**：角丸正方形・黒地。上「revo」（Jost 200・白）＋下「金帯にサービス名」（Jost 500・黒）。全画面で統一。
- **ワードマーク**：`revo `（白）＋ サフィックス `fit`/`life`（金）。タグライン「revo in your life」。
- デザイントークン（色・角丸・タイポ）は各ファイル内 `T`（テーマ）オブジェクトに集約。

### 価格・課金の設計（重要・確定済み）
- 単体：**無料 / ベーシック ¥580 / スタンダード ¥1,200**（月額）。
- **横断プラン（Cross）＝ fit + life を2つまとめて ¥1,960/月**（定価¥2,180の約10%オフ）。年払い **¥19,600/年**（月額×10＝2ヶ月分無料）。
- 割引率の思想：**2アプリで10%オフ、3アプリで15%オフ**。running 投入時に「3つ横断」が登場する設計（現在は running 未発売のため 2つ横断が実売）。
- **「今後の新アプリも追加費用なし」という約束はしない**（撤回済み）。横断特典は「新しいアプリも、登場したら横断に追加できます」（拡張性のみ示す・無料は約束しない）。
- **14日間の全額返金**（新規購入のみ）：14日以内は全額返金／以降は解約しても更新日まで利用可・返金なし／**更新〈2年目以降〉には14日返金を適用しない**（前受金保護）。※日数・各国消費者保護法の適用は要・弁護士確認。
- 年払いの見せ方：月払い年額を取り消し線で並べ、「◯円お得（2ヶ月分無料）」を金額で明示＋「まとめてお支払いの分、お返しします」。
- **世界対応**：全通貨のサンプル価格が各画面の辞書に入っている（¥ / $ / € / ₩ / ฿ / R$ / ₹ / ₫ / Rp / NT$ など）。
- 割引タグは **言語ごとに最適化**（日本語=まとめ割 / English=SAVE / 中文=套餐 / 한국어=묶음 / 欧州系=PACK/SET/COMBO 等）。数字（-10%/-15%）は共通。

### world（別格・特別枠）
- **revo world はゲーム要素（アバターが世界を歩くメタバース）を持つ別格の存在**。開発・運用コストが桁違いのため、**横断プランには含めない**。
- UI上は名前を出さず「**?**（構想中・未定）」で表示。ローンチ時のサプライズを狙う演出（名前を伏せる）。
- 横断プランの内訳・特典から world の名前は削除済み。

### 領収書（receipt）
- **二段構え**：①支払い明細（消費者向け・デフォルト。税込内訳・但し書き）②正式な領収書（事業者・個人事業主向け。宛名・登録番号入り／インボイス対応）。
- 税額・登録番号・発行者情報は**サンプル表示**。正確な計算・各国税制対応は要・税理士確認（画面内に明記済み）。

## 4. 未実装（Claude Code 側で作るもの）

以下はモックには入っていません（UIの想定のみ）。実装が必要です。

- **決済**（Stripe 等）：サブスク作成・更新・解約・**日割り/返金ロジック**・14日返金・前受金会計。※ アップロード元に `page.tsx`（Next.js + Supabase + Stripe の入口）と `schema.sql`（DB 25テーブル）あり。それを基盤に。
- **認証・アカウント**：ログイン、アカウント連携（fit/life/running を1つの revo アカウントで束ねる）。
- **DB連携**：全画面のサンプルデータを実データに。
- **画面遷移**：各モックは単体HTML。ボタン押下は多くが「トースト表示（プレビュー）」で止めてある。実際の画面遷移・ルーティングに置き換える。
  - 例：plans/billing の「横断を選ぶ」→ cross_confirm へ／cross_confirm 完了 → score_total へ／billing 履歴の「領収書」→ receipt へ。
- **領収書PDF生成**・インボイス登録番号の実発行。
- **アフィリエイト／広告配信**の実基盤、AB テスト基盤。
- **多通貨の実価格**：辞書の値は目安。実際の各国価格・税率は要確認。

## 5. 実装時の注意（設計の意図を壊さないために）

- **価格の整合**：横断は全画面 **¥1,960（月）/ ¥19,600（年）** で統一済み。変更時は plans / cross_confirm / billing / receipt の4画面を必ず揃える。
- **world は名前を出さない**：「?・構想中」を維持。ローンチまで名前を伏せる演出。
- **品位（ブランドの核）**：広告で急かさない・記録中や頑張る瞬間に広告を出さない。ランキングは現地ユーザー投稿で信頼を守る。この思想を実装でも保つ。
- **誠実さ**：課金日・金額・解約・返金を隠さず明示する設計。ダークパターンを入れない。
- **間口を狭めない**：無料・¥580 の入口を必ず残す（世界中の誰でも始められる）。ミッション「知恵を収入に変える場」。
- **医療・法務の免責**：体組成・栄養・税・消費者保護に関する箇所は専門家確認前提の注記が入っている。消さない。
- **プラン選択のメイン導線**は階段型の `plans`。`builder`（組み立て画面）は横断を検討する人向けの**補助**。混在させると料金体系が複雑化するため、メイン/補助の役割を保つ。
